# Student Hub Portal Framework v2.0

A premium, modern Full-Stack Web Application designed to streamline campus resource tracking. This platform features an administrative dashboard and user lobby where students can seamlessly browse industrial internship attachments and monitor off-campus hostel accommodations.

Built with a fast, lightweight architecture, it utilizes a custom glass-morphism dashboard UI paired with a responsive database backend.

## 🎯 Key Features

- **Dual-Role Authentication:** Dynamic state-switching authentication lobby supporting secure account creation and login for both **Students** and **System Administrators**.
- **Internship Placement Desk:** Live database management panel to view, add, or delete corporate attachment details, complete with focus fields, location tags, and direct HR contact pipelines.
- **Accommodation Matrix:** Live catalog structure tracking local campus hostels, pricing matrices, and room specifications.
- **Dynamic Frontend Engine:** Built with a clean JavaScript State Machine controller layer, high-performance Tailwind CSS elements, and responsive layout micro-interactions.
- **Robust Storage Architecture:** Uses a reliable SQLite database schema mapping structural entity models with error exception-handling loops.

## 💻 Tech Stack

- **Frontend:** HTML5, JavaScript (ES6+), Tailwind CSS (via CDN), Google Fonts (Plus Jakarta Sans)
- **Backend:** Node.js, Express.js
- **Database:** SQLite3
- **File Management:** Multer Engine Middleware
